# XCLocation

这是一个仿照微信定位的小功能demo。适用于即时通讯项目中发送地理位置，普通App中定位检索附近位置等；

具体功能包括：
 - 定位到当前位置
 - 可定位当前位置附近的Poi
 - 可自行搜索目标位置，并展示附近的poi
 - 选择当前位置或者选择目标位置进行发送
